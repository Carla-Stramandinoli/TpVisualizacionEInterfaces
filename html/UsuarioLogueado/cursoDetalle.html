Curso de Tecnología

JavaScript Avanzado

Tecnología - Desarrollo Avanzado
Descripción del curso

Domina conceptos avanzados de JavaScript: closures, prototipos, async/await, y patrones modernos de desarrollo web. Este curso está diseñado para desarrolladores que ya tienen experiencia con JavaScript básico y quieren llevar sus habilidades al siguiente nivel profesional.

Instructor

María González - Senior Developer

Duración

6 semanas (36 horas)

Modalidad

Online en vivo + Grabaciones

Fecha de inicio

15 de agosto, 2024

Horario

Martes y Jueves 19:00 - 21:00

Objetivos del curso
  • Dominar closures, scope y el contexto de ejecución
  • Entender prototipos, herencia y programación orientada a objetos
  • Manejar programación asíncrona con Promises y async/await
  • Aplicar patrones de diseño modernos en JavaScript
  • Optimizar el rendimiento y debugging avanzado
  • Desarrollar un proyecto completo aplicando todos los conceptos
Contenido del curso
  • Contexto de ejecución y call stack
  • Closures: conceptos y casos de uso
  • Scope chain y hoisting avanzado
  • Ejercicios prácticos con closures
  • Prototype chain y Object.create()
  • Clases ES6 vs funciones constructoras
  • Herencia y polimorfismo
  • Mixins y composición
  • Event Loop y Web APIs
  • Promises: creación y manejo
  • async/await y manejo de errores
  • Fetch API y llamadas HTTP
  • Module Pattern y Revealing Module
  • Observer Pattern y Event Emitters
  • Singleton y Factory Pattern
  • Decorator Pattern en JavaScript
  • Profiling y análisis de rendimiento
  • Memory leaks y garbage collection
  • Debugging avanzado con DevTools
  • Testing unitario con Jest
  • Planificación y arquitectura del proyecto
  • Desarrollo de una aplicación completa
  • Code review y buenas prácticas
  • Presentación y feedback
Requisitos previos
  • Conocimientos sólidos de JavaScript básico
  • Experiencia con ES6+ (let/const, arrow functions, destructuring)
  • Manejo básico de HTML y CSS
  • Familiaridad con herramientas de desarrollo (DevTools)
Herramientas que utilizarás
  • Visual Studio Code
  • Node.js y npm
  • Git y GitHub
  • Jest para testing
  • Babel y Webpack
  • Chrome DevTools
